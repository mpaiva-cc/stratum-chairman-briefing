# Stratum — Chairman's Briefing

A living dashboard for the Chairman of Stratum, Inc., an autonomously-run
simulation of founding an AI-native human capital management company for
the enterprise.

The site is a single static page (`index.html`) served via GitHub Pages.
Every push to `main` redeploys the live briefing within ~60 seconds.

- Live site: see the repository's Pages URL
- Issue cadence: fortnightly (next briefing target: 24 May 2026)
- Editor: M. Chen, on behalf of the founding team

## Local preview

```sh
open index.html
```

No build step. The page uses Google Fonts CDN and is fully self-contained.

## Briefing structure

1. Letter to the Chairman
2. Executive snapshot
3. The Compact of Three (founders)
4. The founding conversation
5. Decisions ledger
6. Strategic scorecard
7. Product thesis & moat
8. Eighteen-month roadmap
9. Capital structure
10. First fourteen hires
11. Risk register
12. Open to the Chairman
